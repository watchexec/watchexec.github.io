Release notes

No code changes this release.

  • 📦 You can now install from binary using cargo-binstall on Linux and macOS. Windows support will depend on binstall#37.
  • 📜 For servers, the documentation now recommends systemfd over and above catflap.
  • 🧑‍🤝‍🧑 The project now officially uses Caretaker Maintainership and a blurb about maintenance attitude had also been previously added last year. This may have implications for you, as a user, especially wrt supply chain issues. The commit message has more details.
