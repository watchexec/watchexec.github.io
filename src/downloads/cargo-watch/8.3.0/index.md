# Cargo Watch 8.3.0

## Release notes

<ul dir="auto">
<li>Add <code class="notranslate">--env-file</code> option to read environment variables from file, like a <code class="notranslate">.env</code>. Note that <code class="notranslate">.env</code> files are not automatically loaded, you do need to provide the option. <li>Add <code class="notranslate">-L</code>, <code class="notranslate">-E</code>, <code class="notranslate">--env-var</code> to manpage and completions file. (<a class="issue-link js-issue-link" data-error-text="Failed to load title" data-id="1528962367" data-permission-text="Title is private" data-url="https://github.com/watchexec/cargo-watch/issues/236" data-hovercard-type="pull_request" data-hovercard-url="/watchexec/cargo-watch/pull/236/hovercard" href="https://github.com/watchexec/cargo-watch/pull/236">#236</a>)</li>
